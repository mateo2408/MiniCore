🎓 Explicación del Patrón MVC en este Ejercicio

M

Modelo (Model)

Definido en el backend mediante schemas de Mongoose (Repartidor.js, Zona.js, Envio.js). Administra las estructuras de datos y relaciones de llaves foráneas lógicas.

V

Vista (View)

La interfaz gráfica interactiva construida en Angular HTML (app.component.html + CSS). Presenta los datos formateados con pipes (currency, number, date) de forma atractiva.

C

Controlador (Controller)

Representado en el backend por envioController.js (procesa la lógica de cálculo agrupada) y en el frontend por app.component.ts (gestiona las peticiones HTTP y estados de UI).

🔄 Flujo de Datos: La Vista envía el rango de fechas al Controlador de Angular $\rightarrow$ Éste hace una petición HTTP GET $\rightarrow$ El Controlador Express del Backend recibe la fecha, consulta los datos al Modelo Mongoose $\rightarrow$ Realiza el cálculo (peso * tarifa) $\rightarrow$ Retorna un JSON $\rightarrow$ La Vista se actualiza reactivamente mostrando los resultados.
